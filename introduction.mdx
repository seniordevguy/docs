# What is Recall?

Recall is a self-hosted AI platform designed for teams and enterprises. It provides a unified interface for interacting with multiple AI models while layering on enterprise-grade features like knowledge injection, persistent memory, tool execution, and custom agent profiles.

## Architecture

Recall is a Next.js application backed by a PostgreSQL database. It uses the Vercel AI SDK for streaming model responses and supports any OpenAI-compatible API endpoint.

**Key components:**

- **Chat Engine** — Streaming AI responses with multi-step tool calling (up to 25 steps per turn)
- **Knowledge System** — Always-inject and semantic retrieval knowledge with org/group/user scoping
- **Skill Framework** — Built-in and installable skills with per-org configuration and encryption
- **Agent Profiles** — Reusable AI personas with model overrides and skill filtering
- **Tracing** — Full request tracing with tool call spans for observability
- **PII Detection** — Automatic redaction of SSNs and credit card numbers
