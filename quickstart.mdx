# Quickstart

Get up and running with the Recall API in under 5 minutes.

## 1. Create an Account

Sign up at your Recall instance and create or join an organization.

## 2. Generate an API Key

Navigate to **Settings → API Keys** and create a new key:

```bash
curl -X POST https://your-instance.com/api/keys \
  -H "Content-Type: application/json" \
  -H "Cookie: your-session-cookie" \
  -d '{
    "name": "My First Key",
    "expiresInDays": 90
  }'
```

<Warning>
  The full API key is returned **only once** in the response. Store it securely — it cannot be retrieved again.
</Warning>

## 3. Make Your First Chat Request

Send a message to the AI using your API key:

```bash
curl -X POST https://your-instance.com/api/chat \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer rk_your_api_key_here" \
  -d '{
    "messages": [
      {
        "role": "user",
        "content": "What can you help me with?"
      }
    ]
  }'
```

The response is a **Server-Sent Events (SSE) stream**. Each chunk contains part of the AI's response, tool calls, and usage metadata.

## 4. Create a Conversation

To maintain context across messages, create a conversation and pass its ID:

```bash
# First message creates context
curl -X POST https://your-instance.com/api/chat \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer rk_your_api_key_here" \
  -d '{
    "messages": [
      {"role": "user", "content": "My name is Alex. I work on the backend team."}
    ],
    "conversationId": "conv_abc123"
  }'

# Follow-up message has full context
curl -X POST https://your-instance.com/api/chat \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer rk_your_api_key_here" \
  -d '{
    "messages": [
      {"role": "user", "content": "What team did I say I work on?"}
    ],
    "conversationId": "conv_abc123"
  }'
```

## 5. Select a Model

Specify a model using the `provider/model` format:

```bash
curl -X POST https://your-instance.com/api/chat \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer rk_your_api_key_here" \
  -d '{
    "messages": [
      {"role": "user", "content": "Explain quantum computing briefly."}
    ],
    "model": "anthropic/claude-sonnet-4-20250514"
  }'
```

Default model is `openai/gpt-4o` if not specified.
